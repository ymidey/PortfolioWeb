Crystal'Viz

Octobre 2023

Visiter le site

Description du projet

Le projet Crystal-Viz, est un projet de troisième semestre de BUT MMI réalisé par groupe de 3 étudiants. Il se présente sous la forme d'une interface web qui, à l'aide d'une représentation visuelle interactive permet de visualiser de manière claire et attrayante les données liées à l'évolution des pays ayant remporté le Cristal du Long Métrage au Festival d'Annecy au fur et à mesure des années. Lors de ce projet, j'ai été en charge du développement web ainsi que de la récolte de données.

Technologies utilisées

Language de programmation :

  • HTML
  • CSS
  • Javascript

Librairies :

  • GreenSock
  • D3.JS

Ce que j'ai appris

Ce projet m'a permis d'apprendre à développer un site web en intégrant des représentations graphiques de données en utilisant la librairie D3.js. Il m'a également permis d'explorer les techniques d'animation à l'aide de la librairie GreenSock.
