Description du projet

Le projet Crystal-Viz, est un projet scolaire réalisé par groupe de 3 étudiants de BUT MMI. Il se présente sous la forme d'une interface web qui, à l'aide d'une représentation visuel interactive qui permet de visualiser de manière claire et attrayante les données liées à l'évolution des pays ayant remporté le Cristal du Long Métrage au Festival d'Annecy.
