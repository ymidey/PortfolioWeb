Crystal'Viz

Octobre 2023

Projet scolaire réalisé en groupe de 3 personnes

Voir le projet

Description du projet

Le projet Crystal-Viz, est une interface web qui, à l'aide d'une représentation visuel interactive qui permet de visualiser de manière claire et attrayante les données liées à l'évolution des pays ayant remporté le Cristal du Long Métrage au Festival d'Annecy.

Technologies utilisées

Language de programmation : HTML - CSS - Javascript
Librairie : GSAP, D3.JS
