Le Donjon du Kraken

Juin 2023

Visiter le site

Description du projet

Le Donjon du Kraken est un projet universitaire réalisé en première année de BUT MMI consistant à la réalisation d'un site proposant une option de réservation de produits. Étant passionné par les jeux de société en tout genre, j'ai conçu un site web permettant aux utilisateurs de réserver des places pour participer à des événements organisés par une boutique fictive nommée Le Donjon du Kraken. Grâce à la mise en place d'une session utilisateur, le site offre la possibilité de réserver plusieurs événements simultanément. De plus, un courriel de confirmation est automatiquement envoyé à l'utilisateur dès la validation de sa réservation.

Technologies utilisées

Language de programmation :

  • HTML
  • CSS
  • Javascript
  • PHP

Ce que j'ai appris

Grâce à la réalisation de ce projet, j'ai appris, à l'aide de PHP, à intéragir avec des données situées dans une base de données MySQL. J'ai également appris à manipuler le DOM afin de garder en mémoire le panier du visiteur lorsqu'il navigue entre les différentes pages
