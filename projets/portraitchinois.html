Description du projet

Le projet portrait-chinois, est un projet scolaire de première année de BUT MMI réalisé seul. Il se présente sous la forme d'un site web présentant mes goûts et ma personnalité à l'aide d'identification à une personne, un objet, un lieu ou tout autre éléments

Technologies utilisées

Language de programmation :

  • HTML
  • CSS
  • Javascript

Ce que j'ai appris

Grâce à ce projet, j'ai appris à traiter des données JSON avec JavaScript, à développer une modale qui peut s'afficher ou se cacher à l'aide d'un clique sur un bouton, ainsi que l'animation de texte à l'aide de keyframes CSS
